## Generic Template Rule

The generic domain templates:

- `Personal`
- `Social`
- `Experiential`
- `Commercial`

should carry no default cover. A concrete project or goal created from those templates can receive a unique cover later.

## Format Rule

- prefer `.jpg` for photographic covers
- keep names lowercase and hyphen-separated
- keep names semantic and stable so Notion skill logic can reference them directly
